## Installation

### Option 1: Using pip

```bash
pip install -r requirements.txt
```

### Option 2: Using conda

```bash
conda create -n subsystem5 python=3.10
conda activate subsystem5
pip install -r requirements.txt
```

### Option 3: GPU Support (NVIDIA)

```bash
# Install CUDA and cuDNN first
# Then install TensorFlow GPU
pip install tensorflow[and-cuda]>=2.13.0
```
